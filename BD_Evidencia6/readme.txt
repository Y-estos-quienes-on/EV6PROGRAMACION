Instrucciones de Ejecución
Motor de Base de Datos: MySQL 8.0+

Opción 1: OneCompiler (Recomendado)
URL: https://onecompiler.com/mysql
Pasos:

1. Acceder al link (ya tiene MySQL seleccionado por defecto)
2. Copiar y pegar el contenido completo del archivo script_smarthome.sql
3. Hacer clic en el botón "Run" (botón rojo, arriba a la derecha)
4. Ya se pueden visualizar los resultados a la derecha.
